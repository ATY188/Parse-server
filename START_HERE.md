# 🚀 從這裡開始！

你現在有兩個完整的網頁內容解析器 API！

## ⚡ 超快速開始

### 選項 1：Python 版本（推薦給你）🐍

```bash
# 1. 安裝
pip install -r requirements.txt

# 2. 啟動
python parser-server.py

# 3. 測試（開新終端機）
python test-parser.py
```

✨ **特別功能：** 瀏覽器打開 http://localhost:3000/docs 查看自動生成的 API 文件！

---

### 選項 2：JavaScript 版本 🟨

```bash
# 1. 啟動（套件已安裝）
npm start

# 2. 測試（開新終端機）
npm test
```

---

## 🤔 還在猶豫？

**我建議你用 Python 版本**，因為：
- ✅ 你說你會一點 Python
- ✅ 程式碼更容易看懂和修改
- ✅ FastAPI 很現代化，自動生成 API 文件
- ✅ 解析品質很好
